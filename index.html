▶ Tube Finder Search
Uses the YouTube Data API v3. Your API key is stored only in this browser.

Settings

50 is the default. Requesting more than 50 uses additional API quota (one unit of quota per 50 results, roughly 100 quota units each).

Enter your YouTube API key

Tube Finder needs a YouTube Data API v3 key to search. It's saved only in your browser's local storage — it never leaves your device except to talk directly to Google's API.

Getting a YouTube API key

  1. Go to the Google Cloud Console and sign in with any Google account.
  2. Click the project dropdown at the top and choose New Project. Give it any name (e.g. "Tube Finder") and click Create.
  3. With that project selected, open the navigation menu and go to APIs & Services → Library.
  4. Search for YouTube Data API v3, open it, and click Enable.
  5. Go to APIs & Services → Credentials, click Create Credentials, and choose API key.
  6. Copy the generated key and paste it into Tube Finder.
  7. Optional but recommended: click Edit API key and under API restrictions limit it to YouTube Data API v3 so the key can't be used for anything else.

The free tier includes 10,000 quota units per day. A single search costs about 100 units, so you get roughly 100 searches a day at no cost.
